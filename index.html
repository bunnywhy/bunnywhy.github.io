Homeworks, CS 598 Deep Learning

The course used Python and PyTorch. I learned main structures like CNN, RNN, LSTM, etc. and main applications like object detection and NLP.

HW 1 - Implemented and trained a neural network from scratch in Python for the MNIST dataset (no PyTorch). Test accuracy 97.20%.
HW 2 - Implemented and trained a CNN from scratch in Python for the MNIST dataset (no PyTorch).
Implemented the details of convolution without using SciPy's convolution function. The CNN has one hidden layer and 5 channels. Test accuracy 98.00%.
HW 3 - Trained a deep convolution network on GPU with PyTorch for the CIFAR10 dataset.
The CNN uses dropout and data aumentation, and it was trained with ADAM. Test accuracy 80.09%.
HW 4 - Implemented building blocks of ResNets and put them together to build and train a deep RNN for image classification on CIFAR100. Utilized transfer learning; used a pretrained ResNet and trained it on CIFAR100. Test accuracy without using pretrained model is 63.92%, otherwise 87.95%.
HW 6 - A tutorial on GAN
Trained a GAN for CIFAR10 classification; trained a generator-discriminator pair on CIFAR10 dataset utilizing techniques from ACGAN and Wasserstein GANs.
HW 7&8 - Tutorials on NLP
Trained models to detect the sentiment of IMDB movie reviews. Fed a review into a model and have it to predict whether the review is positive or negative. [reference]
